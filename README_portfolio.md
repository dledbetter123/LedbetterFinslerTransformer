What if attention is not a mechanism you compute, but a curvature you move through? The Finsler Transformer replaces the transformer's O(T²) attention with geodesic flow on a learned Finsler manifold. Instead of scoring every pair of tokens, the model learns a geometry in which a sentence is a single trajectory — a geodesic — and "attention" becomes the cumulative deformation of space the path experiences along its route. The target is an O(T) autoregressive generator whose context-handling lives in the metric rather than being recomputed each step.

Why Finsler? Standard attention is implicitly Euclidean — symmetric and direction-independent — but language is directional: "A implies B" is not "B implies A." A Riemannian metric varies with position but stays symmetric in direction and tends to over-smooth. Finsler geometry uses a direction-dependent norm F(x,v) ≠ F(x,−v). The project's Randers metric F = √(aᵢⱼyⁱyʲ) + bᵢyⁱ learns a background geometry a plus a "wind" 1-form b that biases flow toward the next logical token — the asymmetry language needs, built into the space itself. As the state moves through a sentence, the connection distorts the tangent space for the next token: attention is curvature, and path-dependent meaning is holonomy — different routes to the same point yield different representations, which standard position-only attention cannot express.

It's an honest exploration repo. Synthetic 2D experiments validate the metric-first design (and correctly plateau at the noise floor on stochastic walks); the failure analysis — ungrounded MLP sprays, additive-vs-frame holonomy, loss-space mismatches — is what drove the architecture toward learning the metric explicitly and deriving the geodesic spray analytically via autograd. The current frontier is metric expressiveness: carving the sharp local channels language needs via a hybrid analytic-plus-residual manifold and multi-chart geometries.

Sister project to Sparse Geometric Signal Transport: both treat sequence processing as transport on a learned geometry — SGST in the spectral fiber bundle, Finsler on the base manifold itself. Together they sketch a path-dependent, sub-quadratic generative model grounded in differential geometry.
